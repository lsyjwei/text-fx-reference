# CSS Text FX Reference Collection

CodePen / 第三方 CSS 字效样例归档，供《告亡者》大坟场 Demo 序章字效选型与移植参考。

> 原始 zip 来自本机 Downloads；各包内保留 `LICENSE.txt`（如有）与 `dist/` 可运行预览。

## 包清单

| 目录 | 来源 zip | 说明 |
|------|----------|------|
| `packages/glitch-noise-clip` | glitch-text-effect (2).zip | **载入行推荐** · 单 keyframe clip 切片 + 双色 shadow |
| `packages/glitch-clip-rgb` | glitch-text-effect.zip | 经典 RGB 色差 + 双 keyframe `clip: rect()` |
| `packages/glitch-rubik-spray` | glitch-text-effect (1).zip | Rubik Spray Paint 字体 + 上下半片错位 |
| `packages/text-glitch-exo` | text-glitch-effect.zip | 轻量 Exo 字体 clip glitch |
| `packages/survival-horror-bleeding` | survival-horror-bleeding-titles-css.zip | 标题渗血滴落 + 正文血渍扩散 |
| `packages/svg-knockout-video` | svg-knockout-text-with-video-background.zip | SVG mask 镂空文字透出视频 |
| `packages/glassmorphism-portfolio` | kuldeep-…portfolio.zip | 玻璃拟态 UI（非字效主包） |

`text-glitch-effect (1).zip` 与 `text-glitch-effect.zip` 内容相同，未重复收录。

## 本地预览

各包 `dist/index.html` 可直接用浏览器打开（部分依赖 Google Fonts / CodePen CDN）。
